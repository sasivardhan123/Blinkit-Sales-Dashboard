# Blinkit Sales Dashboard

## Project Overview
This project analyzes grocery sales data of Blinkit using Power BI. 
The dashboard helps understand sales performance, outlet distribution, and product demand.

## Tools Used
- Power BI
- Excel
- Data Visualization

## Key Insights
- Total Sales: $1.20M
- Total Items: 9K
- Average Rating: 4

## Dashboard Features
- Sales by Item Type
- Outlet Size Analysis
- Location-based Sales
- Fat Content Distribution
